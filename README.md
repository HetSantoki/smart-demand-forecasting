# 📦 Smart Demand Forecasting

![Python Version](https://img.shields.io/badge/python-3.10%2B-blue)
![Django Version](https://img.shields.io/badge/django-4.2%2B-green)
![License](https://img.shields.io/badge/license-MIT-blue)

A Django-based web dashboard for forecasting product demand from UK e-commerce 
sales data (Dec 2010 – Dec 2011).

---

## 📄 Overview

**Smart Demand Forecasting** helps businesses predict future product demand,
identify stockout risks, and make smarter inventory decisions — all through
an interactive web dashboard.

### Features
- 📊 **Overview Dashboard** — Historical demand chart with festival markers and forecast
- 🔍 **Product Forecast** — Per-product demand forecasting with custom date ranges
- ⚠️ **Stockout Alerts** — Risk level badges (High / Medium / Low) for top products
- 🌙 **Dark / Light Mode** — Toggle between themes

## 🚀 Quick Start

### Prerequisites
- Python 3.10+
- Git

### Installation

1. **Clone the repository**
```bash
   git clone https://github.com/YOUR_USERNAME/Smart-Demand-Forecasting.git
   cd Smart-Demand-Forecasting
```

2. **Create a virtual environment & install dependencies**
```bash
   python -m venv venv
   
   # Windows
   venv\Scripts\activate
   
   # Mac/Linux
   source venv/bin/activate
   
   pip install -r requirements.txt
```

3. **Apply database migrations**
```bash
   cd forecast_project
   python manage.py migrate
```

4. **Run the development server**
```bash
   python manage.py runserver
```
   Open http://127.0.0.1:8000 in your browser.

## 📊 How It Works

1. **Historical data** is loaded and aggregated into daily demand
2. **Forecasting model** predicts future demand for selected date range
3. **Stockout alerts** are generated by comparing recent vs average demand
4. **Charts** are rendered using Chart.js with festival markers
